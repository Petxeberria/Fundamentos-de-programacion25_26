1. Creación del fichero de ejercicios y grupos musculares

Para crear el fichero CSV con los ejercicios, grupos musculares y tiempos estimados, utilicé la herramienta de inteligencia artificial ChatGPT. Inicialmente generé la lista de ejercicios por grupo, luego agregué tiempos estimados por ejercicio (5-12 minutos) para hacer el cálculo de rutinas más preciso.

Prompt inicial: "Genera un archivo CSV con ejercicios de gimnasio para los grupos musculares: hombro, biceps, triceps, pecho, espalda, pierna, abdominales. Mínimo 3, máximo 5 por grupo. Columnas: ejercicio, grupo_muscular."

Prompt para tiempos: "Agrega una columna 'tiempo_minutos' al CSV anterior con tiempos estimados realistas para cada ejercicio (5-15 minutos)."

Luego, copié la salida y la guardé en ejercicios.csv.

2. Creación del script de Python

El script rutina_gym.py fue desarrollado usando Python básico, sin programación orientada a objetos como se indicó. Utilicé las siguientes herramientas y pasos:

- Librerías: csv para leer el archivo, os para manejar directorios y archivos, random para generar la rutina aleatoriamente.

- Funciones creadas: cargar_ejercicios (lee CSV con tiempos), crear_rutina (genera rutina con cálculo de tiempos por ejercicio), guardar_rutina (guarda en txt), cargar_rutina (lista y muestra guardadas), menu (interfaz principal).

- Para la generación de rutinas: Mezcla los grupos y los distribuye en días usando round-robin para asegurar que todos los grupos estén incluidos y cada día tenga al menos uno. Calcula tiempos sumando los minutos de cada ejercicio seleccionado, y ajusta quitando ejercicios de mayor tiempo si excede el límite por sesión.

- Cambios realizados: Inicialmente usé tiempos por grupo, pero el usuario pidió tiempos por ejercicio, así que actualicé el CSV y el script para leer y calcular tiempos individuales, permitiendo un cálculo más preciso de si la rutina cabe en el tiempo especificado.

- Modelo de IA usado: ChatGPT para generar el contenido del CSV con tiempos.
